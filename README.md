## Key Insights
- Alcohol is the strongest predictor of wine quality
- Volatile acidity negatively impacts quality
- Most wines fall within mid-range quality scores (5–7)
- Random Forest achieved the best predictive performance

---

## Features
- KPI indicators for quick insights
- Correlation analysis and feature relationships
- Interactive filters (wine type, quality)
- Machine learning model comparison
- Feature importance and key influencers analysis

## Tools Used
- Power BI
